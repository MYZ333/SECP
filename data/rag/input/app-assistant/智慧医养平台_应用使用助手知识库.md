# 智慧医养平台普通用户功能使用说明

来源机构：智慧医养平台产品组
官方来源：智慧医养平台 Web 端内部产品实现（暂无公开访问链接）
资料类型：平台功能说明
适用对象：智慧医养平台普通用户、老年用户及协助其操作的家属
适用终端：普通用户 Web 端
文档版本：2026-07-15

## 1. 文档用途与回答边界

本文档是“应用使用助手”的功能知识库，只用于回答普通用户怎样使用智慧医养平台 Web 端。应用使用助手可以解释页面入口、按钮位置、填写方法、操作结果和常见错误，但不能替用户点击按钮、修改数据、兑换商品、结束咨询或注销账号。

应用使用助手不提供疾病诊断、用药方案、检查结果判断或个体化就医建议。用户询问症状、疾病、药物、检查报告或健康风险时，应引导用户进入顶部菜单“健康助手”咨询；如情况紧急，应及时联系线下医疗机构或急救服务。

本文档只介绍普通用户可见功能，不提供管理员后台、医生工作台、知识库管理、用户管理、商品管理或医生审核等操作说明。普通用户找不到“系统管理”等菜单属于正常现象。

任何助手都不会向用户索取登录密码、短信验证码、身份证号码或银行卡密码。用户不要在聊天窗口发送密码或验证码。

## 2. Web 端页面与导航

登录后页面顶部有“首页、健康档案、积分中心、医生专家库、健康助手、医生咨询、健康预警、账户管理”等菜单，顶部还有“AI 助手”按钮。本文所有入口和步骤均以普通用户 Web 端为准。

“健康档案”和“积分中心”是下拉菜单。将鼠标移到“健康档案”可看到基本信息、体征数据、就诊记录、健康报告和健康时间轴；将鼠标移到“积分中心”可看到积分商城和积分明细。点击具体子项进入页面。普通用户看不到“系统管理”属于正常现象。

页面右上角显示用户头像和昵称。点击可进入“账户管理”或“退出登录”；将鼠标停留在头像区域可查看今日健康摘要。

## 3. 注册、登录与退出

### 3.1 Web 端用户名密码登录

入口：打开 Web 登录页，默认显示“账密登录”。

操作步骤：

1. 在“请输入用户名”中填写注册时使用的用户名。这里填写的是用户名，不是昵称。
2. 在“请输入密码”中填写密码，可使用输入框中的显示密码按钮核对。
3. 点击“立即登录”，或在密码框中按 Enter。
4. 登录成功后进入首页。

用户名和密码不能为空。连续提示“用户名或密码错误”时，应检查大小写、空格以及是否误把昵称当成用户名。应用使用助手不能查询或显示用户密码。

### 3.2 Web 端手机号验证码登录

入口：登录页右侧切换到“手机号登录”。

操作步骤：

1. 输入中国大陆 11 位手机号。
2. 点击“获取验证码”。同一手机号 60 秒内不能重复发送。
3. 输入收到的 6 位验证码。
4. 点击“立即登录”。

验证码有效期为 5 分钟，验证成功后立即失效。同一手机号每天最多请求 10 次验证码；验证码连续输错达到限制后会失效，需要重新获取。未注册的手机号验证成功后会自动创建普通用户账号。

手机号自动创建的账号由系统生成随机用户名和随机初始密码。用户可在“账户管理”查看系统生成的用户名；后续可以继续使用手机号验证码登录。如果希望使用用户名密码登录，可先退出，在 Web 登录页通过“忘记密码”用该手机号设置一个自己知道的新密码，再用账户管理中显示的用户名登录。

如果页面提示验证码已发送但手机没有收到，可能是当前部署环境未接入真实短信服务，应联系平台工作人员；不要把验证码告诉应用助手。

### 3.3 Web 端注册普通用户账号

入口：登录页点击“前往注册”。

操作步骤：

1. 填写 3 至 20 位用户名。用户名用于以后账密登录，注册后页面不提供修改用户名的功能。
2. 填写 6 至 32 位密码。
3. 按需填写昵称和手机号。昵称、手机号可以稍后在“账户管理”中修改。
4. 点击“立即注册”。
5. 注册成功后返回账密登录，使用刚才的用户名和密码登录。

用户名必须唯一。若提示用户名已存在，应更换用户名。手机号虽在注册表单中标为可选，但建议填写常用手机号，便于以后通过验证码重置密码。

### 3.4 Web 端忘记密码

前提：账号已经绑定可正常接收验证码的手机号。

入口：登录页点击“忘记密码”。

操作步骤：

1. 输入注册手机号。
2. 点击“获取验证码”，填写 6 位验证码。
3. 输入 6 至 32 位新密码。
4. 点击“重置密码”。
5. 页面提示成功后，返回账密登录并使用新密码。

如果账号没有绑定手机号、原手机号已停用或始终收不到验证码，当前页面无法自行找回，应联系平台工作人员核验处理。

### 3.5 登录页大字号和二维码说明

Web 登录页右上角可切换“大字号”和“小字号”，只影响当前登录页显示。

登录页左侧的二维码目前是界面展示图，项目中尚未接入真实扫码登录流程。普通用户应使用账密登录或手机号验证码登录，不要把二维码展示区当成已开通的登录方式。

### 3.6 退出登录

点击页面右上角头像或用户名，在下拉菜单中点击“退出登录”。

退出只结束当前设备的登录状态，不会删除账号和健康数据。再次使用时可以重新登录。

## 4. 首页

### 4.1 首页能看到什么

登录后默认进入“首页”。欢迎区显示日期和累计体征记录数，并提供“记录体征”和“向 AI 咨询”两个快捷按钮。

首页统计卡包括：

- “我的积分”：当前可用积分，点击进入积分明细。
- “体征记录”：累计体征记录数量，点击进入体征数据。
- “健康报告”：累计健康报告数量，点击进入健康报告。
- “健康预警”：累计预警记录数量，点击进入健康预警。

统计卡旁的“7 日 +数字”表示近 7 日新增数量。首页下方还有健康档案、体征数据、积分商城、医生专家库、健康咨询和健康预警等快捷入口。

将鼠标停在右上角头像区域，可查看“今日健康摘要”，包括今日体征记录数、预警数和当前积分。

首页和头像摘要中的预警数字会随当前预警概况更新。顶部可能弹出“健康预警已更新”通知，点击通知可进入健康预警页面。预警是否已经处理，应以“健康预警”页面中的待处理、处理中、已解决等状态为准。

## 5. 应用使用助手

### 5.1 应用使用助手能做什么

应用使用助手专门回答“平台怎么用”，例如：在哪里记录血压、如何修改资料、积分怎样兑换、怎样找医生、预警在哪里查看。它不读取健康助手的医疗知识来诊断疾病，也不能代替用户执行页面操作。

入口：登录 Web 端后，点击顶部“AI 助手”。右侧会打开标题为“SECP-AI助手”的面板。

### 5.2 如何提问

用户可直接输入问题，也可点击平台使用指南、健康档案、体征数据、健康预警、积分与兑换、医生服务等快捷问题。按 Enter 发送，Shift + Enter 换行。

建议在问题中直接说明目标，例如：“怎样新增血压？”“怎么查看兑换记录？”

### 5.3 应用使用助手的当前限制

- 面板关闭后，当前可见的消息会被清空；界面没有应用助手历史会话列表。重要步骤应自行记录。
- “深度思考”按钮目前只改变按钮选中状态，当前请求没有单独的深度思考参数，不应期待它切换到另一种回答模式。
- 应用助手不会替用户打开页面、保存资料、兑换商品或注销账号。
- 若知识库没有收录某项功能，助手应明确说明无法确认，不能编造按钮或入口。

### 5.4 应用使用助手和健康助手的区别

问“怎么操作、按钮在哪、为什么找不到入口”时使用顶部“AI 助手”。

问“血压高怎么办、这种药怎么吃、检查结果是什么意思、是否需要就医”时使用顶部“健康助手”。

应用使用助手回答平台使用问题；健康助手回答健康咨询问题。二者入口、知识库和聊天内容相互独立。

## 6. 健康档案基本信息

### 6.1 基本信息包含哪些内容

健康基本信息包括身高、体重、BMI、血型、过敏史、家族病史、既往病史、紧急联系人、紧急联系电话和备注。

BMI 由系统根据身高和体重自动计算，用户不需要手工填写。身高或体重缺失时无法显示 BMI。

健康基本信息不同于“账户管理”中的个人资料。健康基本信息记录健康相关内容；账户管理记录昵称、头像、手机号、性别和生日。

### 6.2 查看和修改健康基本信息

路径：顶部“健康档案” → “基本信息”。

操作步骤：

1. 页面先以只读方式显示当前健康信息。
2. 点击右上角“修改信息”。
3. 填写或修改身高、体重、血型、病史、紧急联系人等内容。
4. 点击“保存”。
5. 页面提示“保存成功”后，新信息生效。

身高可填 0 至 250 厘米，体重可填 0 至 300 千克；过敏史、家族病史、既往病史和备注每项最多 500 字。没有相关病史时可以留空，也可填写“无”。

### 6.3 健康档案与积分任务

Web 积分商城中有“一次性”的“完善健康档案”任务。要满足任务条件，身高、体重、血型、紧急联系人和紧急联系电话必须全部填写。保存后任务会变为“待领取”；还需要回到“积分中心” → “积分商城”手动点击“待领取”，才会获得 20 积分。只保存档案不等于已经领取积分。

## 7. 体征数据

### 7.1 支持哪些体征

平台支持血压、血糖、心率、体温和体重五类记录。

建议单位：血压使用 mmHg，血糖使用 mmol/L，心率使用“次/分”，体温使用 ℃，体重使用 kg。Web 端选择指标类型后会自动带出常用单位，用户仍可修改单位。

血压有两个数值：第一个是收缩压，第二个是舒张压。其他指标只需填写一个主要数值。测量时间不填时，Web 后端默认使用当前时间。

### 7.2 新增体征记录

路径：顶部“健康档案” → “体征数据”，或首页点击“记录体征”。

操作步骤：

1. 点击“新增体征记录”。
2. 选择指标类型。
3. 填写数值；选择血压时分别填写收缩压和舒张压。
4. 核对单位，可选择测量时间并填写备注。
5. 点击“保存”。

保存后系统会按内置阈值自动标记“正常”或“异常”。这只是健康提示，不是医学诊断。

### 7.3 查看体征趋势

“体征数据”页顶部是“体征趋势”。点击血压、血糖、心率、体温或体重标签可切换指标。

同一指标至少需要 2 条记录才能绘制趋势；不足 2 条时会显示“暂无法绘制趋势”。趋势图最多读取最近 30 条该类记录。红点表示被系统标记为异常的记录，鼠标悬停在数据点上可查看测量时间、数值和单位。血压图中实线表示收缩压，虚线表示舒张压。

### 7.4 修改或删除体征

在体征列表找到目标记录，点击“编辑”可修改，保存后系统会重新判断是否异常；点击“删除”并确认可删除。

删除后无法在普通用户页面恢复。若只是数值填错，优先使用“编辑”，不要先删除。

### 7.5 系统判定异常的当前规则

以下规则只用于解释页面为何显示“异常”，不构成诊断：

- 血压：收缩压达到 140 或舒张压达到 90 会标记异常；收缩压达到 160 或舒张压达到 100 时级别更高；收缩压低于 90 或舒张压低于 60 也会标记异常。
- 血糖：低于 3.9、达到 7.0 或达到 11.1 会按不同级别标记异常。系统当前没有区分空腹、餐后或随机血糖，用户应结合实际测量场景理解。
- 心率：低于 60 或高于 100 会标记异常；低于 50 或高于 120 时级别更高。
- 体温：低于 36.0 或达到 37.3 会标记异常；达到 39.0 时级别更高。
- 体重：当前只记录数值，不自动按阈值判断异常。

单次异常可能受测量姿势、设备、时间或录入错误影响。需要健康建议时应进入“健康助手”或咨询医生。

### 7.6 记录体征后的积分

Web 端每天完成一次体征记录后，“记录健康数据”任务会变成“待领取”。回到“积分中心” → “积分商城”点击“待领取”可获得 2 积分；这是每日任务，同一天重复记录不会重复领取该任务积分。

## 8. 就诊记录

### 8.1 就诊记录包含哪些内容

就诊记录可保存就诊日期、医院、科室、医生姓名、诊断、处方或用药情况以及备注。它用于用户自行整理历史就诊信息，不会自动从医院系统导入。

### 8.2 新增、编辑和删除

路径：顶部“健康档案” → “就诊记录”。

新增步骤：

1. 点击“新增就诊记录”。
2. 选择就诊日期。
3. 填写医院、科室和医生。
4. 填写诊断、处方或用药情况，可补充备注。
5. 点击“保存”。

在列表中点击“编辑”可修改已有记录；点击“删除”并确认可删除。普通用户只能管理自己的就诊记录。

## 9. 健康报告与健康时间轴

### 9.1 报告类型

平台报告类型包括“体检报告”“AI 生成”“其他”。报告记录可包含标题、报告日期、内容或结论，以及附件 URL。

“附件 URL”只是填写已有文件链接的文本框，不是本地文件上传按钮。当前普通用户报告页面没有直接上传体检报告文件的功能，而且“查看”弹窗目前只展示报告内容，不展示或打开附件 URL。

### 9.2 手工新增报告

路径：顶部“健康档案” → “健康报告”。

点击“新增健康报告”，填写标题、类型、报告日期、内容或结论，按需填写附件 URL，然后点击“保存”。保存后可在列表中“查看、编辑、删除”。

### 9.3 生成 7 天、30 天或 90 天分析报告

前提：所选统计周期内至少有 1 条包含有效数值的体征记录。

操作步骤：

1. 进入“健康档案” → “健康报告”。
2. 点击“生成分析报告”。
3. 在“统计周期”中选择近 7 天、近 30 天或近 90 天。
4. 在“报告文案”中选择“AI 解读”或“规则模板”。
5. 点击“生成报告”。生成成功后系统会保存一份类型为“自动分析”的报告，并直接打开报告详情。

医学统计、异常判定和关注等级始终由固定规则计算。开启“AI 解读”时，AI 只负责整理摘要和建议的表达；AI 服务不可用时系统会自动退回规则模板，不影响统计和风险结果。关闭“AI 解读”时直接使用规则模板。

如果提示“所选周期内暂无有效体征记录”，先到“健康档案” → “体征数据”新增记录，或改选覆盖已有记录的更长周期后再生成。短时间内对同一周期重复提交时，系统可能直接返回刚生成的同一份报告，以防重复创建。

### 9.4 查看结构化报告详情

在报告列表点击“查看详情”。新版自动分析报告会显示：

- 综合关注等级：正常、数据不足、轻度关注、建议关注或高度关注。
- 统计周期、有效记录数、数据完整度和生成方式。
- 各类体征的最新值、均值、中位数、范围、异常率和趋势。
- 血压的收缩压与舒张压统计、趋势图、异常记录、报告摘要和行动建议。
- 算法版本和“不能替代医生诊断”的提示。

趋势至少需要 3 条记录，并且覆盖至少 3 个不同测量日；记录不足时仍可生成报告，但会显示“数据不足”且不判断趋势。图表少于 2 个数据点时不会绘制曲线。

旧版报告没有结构化统计数据，查看时会显示“旧版文本报告”提示，并继续展示原有文本内容。手工新增的报告仍可查看、编辑和删除。

分析报告只使用平台中的体征记录和基本健康资料，不会读取用户没有上传到平台的医学影像或检验单，也不构成诊断。

### 9.5 健康时间轴

路径：顶部“健康档案” → “健康时间轴”。

健康时间轴会在一个页面汇总最近的体征、健康预警、健康报告、就诊记录、医生咨询和医生用药建议。页面顶部显示健康闭环分数和状态提示，下方显示体征异常、未读预警、已结束咨询和待确认用药等概况。

在“健康事件流”中可点击“全部、体征、预警、报告、就诊、咨询、用药”筛选事件。每条事件都有“查看来源”或“进入对话”按钮，可返回对应的体征、预警、报告、就诊或医生咨询页面。

健康闭环分数用于提醒是否有未处理预警、异常体征、待完成咨询或待确认用药，不是医学评分或诊断。部分数据接口暂时不可用时，时间轴会尽量展示已经成功读取的内容。

## 10. 健康预警

### 10.1 预警是怎样生成的

保存或修改血压、血糖、心率、体温等受支持的体征后，系统会自动检查该条记录。若记录异常，系统会结合最近 7 天同类体征创建或更新预警，日常使用无需手动点击生成。

同一指标已有未解决预警时，系统会更新这条预警的级别、内容、最近触发时间和触发次数，而不是连续创建多条相同预警。完全相同的一批数据会被去重。预警会显示高危、中度或提示级别，以及新预警、处理中、已解决、已忽略等状态。

### 10.2 查看、筛选和处理预警

路径：顶部“健康预警”。

页面顶部“处理概况”显示待处理、高危待处理、处理中和已解决数量。预警事项可按处理状态和级别筛选。

对待处理或处理中的预警可执行：

- “问健康助手”：带着预警内容进入健康助手。发送问题并建立会话后，该预警会进入“处理中”。
- “咨询医生”：先进入医生专家库选择医生，再进入医生咨询；创建咨询会话后，该预警会进入“处理中”。
- “标记解决”：只在已经复测、获得医生建议或确认问题处理完成后使用，可选填不超过 500 字的处理说明。
- “误录”：只用于数值误录、设备异常或无效记录，必须填写不超过 500 字的忽略原因。

“已解决”和“已忽略”表示本轮预警处理结束，但不会删除原始体征记录。若以后又出现新的同类异常，系统可以再次创建新的预警。

### 10.3 历史数据补偿检测

“重新检测历史体征”只用于规则升级后补查旧记录，或旧体征没有生成预警的情况。点击后系统先预览最近 7 天可分析记录、异常数量和异常类型；确认后点击“同步历史异常”。

补偿检测也会合并同类未解决预警并跳过重复结果。日常新增或修改体征已经会自动触发检测，不需要反复执行历史检测。

### 10.4 预警页面注意事项

- 顶部可能出现新预警通知，点击通知可打开健康预警页面。
- 待处理数量包括新预警和已确认但尚未结束的预警；“处理中”表示已经从健康助手或医生咨询开始跟进。
- 修改异常体征后，系统会重新分析；删除体征不会自动删除已经生成的预警，需要按实际情况标记解决或误录。
- 预警不能代替医生诊断。对预警内容有健康疑问，可转到“健康助手”或在 Web 端发起医生咨询。

## 11. 健康助手

### 11.1 健康助手用于什么

健康助手用于健康科普、症状信息整理、风险提示和一般健康建议。回答仅供参考，不替代医生面诊、检查和诊断。

路径：顶部“健康助手”。

### 11.2 开始新对话

进入健康助手后，可点击示例问题，或在底部输入框输入问题。按 Enter 发送，Shift + Enter 换行。

点击左侧“开启新对话”会清空当前界面并开始新的会话；旧会话仍保存在左侧“历史对话”中。点击历史会话可查看之前的问答。

### 11.3 “结合我的健康档案”

聊天顶部有“结合我的健康档案”开关。开启后，健康助手可在本次咨询中按最小范围读取与问题相关的个人健康资料；关闭后不主动结合个人档案。

回答下方如显示“已参考：某类资料”，表示本次回答使用了相应档案类别。该开关偏好会保存在当前浏览器中，后续进入页面时沿用，用户可随时关闭。

### 11.4 问诊补充与风险提示

对个体症状、病因或个性化建议类问题，健康助手可能先逐次提出补充问题，每次只问一个关键问题。用户可点击快捷选项，也可在文本框自由填写。

如果不清楚，可以回答“不知道”；若不想继续补充，可以回答“直接回答”。信息越完整，回答越有针对性。

回答可能显示风险评估：一般咨询、建议关注、尽快就医或立即就医。风险提示是辅助分流，不是诊断。出现急危重症状时应优先寻求线下急救，不要等待聊天回答。

### 11.5 生成过程和中断

Web 端会显示安全分诊、问诊补充、任务调度、咨询分析、权威检索、整合回答等协作步骤。回答以流式方式逐步显示。

若连接中断但已收到部分内容，页面会保留已收到部分并提示生成中断；用户可以重新提问。发送期间不要重复点击发送按钮。

### 11.6 健康助手推荐医生

健康助手在完成风险判断和回答后，可能根据当前诉求匹配医生。推荐卡会显示医生姓名、职称、医院、科室、擅长领域、匹配度和推荐原因。

点击“向 TA 咨询”可直接进入该医生的咨询页面；点击“查看全部医生”可进入医生专家库继续筛选。匹配度只表示医生资料与本次诉求的相关程度，不代表诊断结论、疗效保证或平台排名。没有合适医生或当前问题不需要医生匹配时，回答下方可能不显示推荐卡。

### 11.7 健康咨询积分任务

Web 端每天使用一次 AI 健康咨询后，“AI健康咨询”任务会变为“待领取”。回到“积分中心” → “积分商城”点击“待领取”可获得 2 积分。发送咨询不等于自动领分。

## 12. 医生专家库

### 12.1 查找医生

路径：顶部“医生专家库”。

可按以下方式查找：

- 在搜索框输入医生姓名、医院或擅长领域关键词。
- 点击科室标签筛选，点击“全部科室”取消科室筛选。
- 在“职称”中选择主任医师、副主任医师或主治医师。
- 点击“搜索”。

医生卡片显示姓名、职称、医院、科室、擅长领域、简介和患者评分。点击卡片或“查看详情”可打开详情，并查看评分及各星级评价占比；点击“向TA咨询”进入医生咨询。暂无评价时页面会明确显示“暂无评价”。

## 13. 医生咨询

### 13.1 发起医生咨询

方式一：顶部“医生专家库” → 找到医生 → 点击“向TA咨询”。

方式二：顶部“医生咨询” → 若尚无会话，点击“前往医生专家库”。

选择医生后，系统会创建或进入与该医生最近的一条咨询会话。左侧显示会话列表，右侧显示消息。再次选择同一医生会进入最近会话；如果该会话之前已结束，系统会将它重新开启。

### 13.2 发送文字

在底部输入想咨询的问题，点击“发送”或按 Enter。医生回复会实时出现在对话中。左侧会话条目可能显示未读数量。

若实时连接短暂断开，页面会自动尝试重新连接。仍无法收到消息时，可刷新页面后重新进入该会话。

### 13.3 发送附件

点击输入区左侧“附件”，选择文件。支持 jpg、jpeg、png、gif、webp 图片，以及 PDF、doc、docx 文档；单个附件不能超过 10MB。

上传成功后附件会作为消息发送。可以先在文字框中填写说明，再选择附件，文字会与附件一起发送。

不要上传与咨询无关的敏感材料。发送前核对文件内容和咨询对象。

### 13.4 查看历史会话

进入顶部“医生咨询”，左侧会列出本人向医生发起的咨询。点击任一会话可查看历史文字和附件。已结束会话仍可查看，但不能继续发送。

### 13.5 结束咨询

在当前会话右上角点击“结束咨询”，阅读确认提示后再次确认。结束后患者和医生都不能继续发送消息，但历史记录仍保留可查。

当前会话结束后不能直接在输入框继续发送。如果仍需咨询，可回到医生专家库再次点击该医生的“向TA咨询”，系统会重新开启最近会话，原历史消息仍保留。

### 13.6 咨询报告与服务评价

会话结束后，医生需要整理并提交咨询报告。报告尚未完成时，患者页面会显示“等待医生报告”；完成后会显示问题概述、初步判断、本次总结、处理建议、风险提醒，以及“建议线下就医”或“居家观察”提示。

咨询报告完成后可提交一次服务评价：

1. 选择 1 至 5 星评分。
2. 按需勾选“回复及时、解释清楚、态度好、建议有帮助”等标签。
3. 可补充不超过 500 字的评价内容。
4. 点击“提交评价”。提交后页面显示“已评价”，当前普通用户页面不能重复修改评价。

医生专家库中的平均评分和评价数量会根据已经提交的评价更新。咨询报告和评价不等同于平台健康报告，入口仍在“医生咨询”的对应会话中。

### 13.7 查看并确认医生用药建议

医生可在咨询结束后发送结构化用药建议。患者在对应的已结束会话中查看药品名称、规格、用法、剂量、频次、天数、注意事项和医生备注。

新建议显示“待确认”。阅读后点击“我已知晓”，状态变为“已确认”。“我已知晓”只表示患者已经看到建议，不代表已经服药、完成购药或同意系统自动执行处方。平台不会自动购买药品，也不会自动把建议写入用户自行维护的就诊记录。

如果会话已经结束但没有显示用药建议，通常表示医生没有为本次咨询发送建议；不要自行根据其他会话的内容用药。对药品名称、剂量或用法有疑问，应继续联系医生或线下医疗机构。

## 14. 积分中心与积分任务

### 14.1 查看积分余额

入口：首页“我的积分”、顶部“积分中心” → “积分商城”，或右上角头像的今日健康摘要。

积分只能用于平台积分商城兑换，不等同于现金，普通用户页面没有积分提现功能。

### 14.2 每日签到

路径：顶部“积分中心” → “积分商城”。

页面顶部显示当前积分、连续签到天数和签到按钮。点击“签到 +数字”即可签到；当天完成后按钮显示“今日已签到”，不能重复签到。

签到基础为 2 分，连续签到每天增加 1 分，连续 7 天及以上每天最高 8 分。签到中断后连续天数重新计算。

### 14.3 积分任务

任务状态有三种：

- “去完成”：尚未满足任务条件，点击可前往相关页面；每日登录任务没有专门跳转页，登录后刷新任务状态即可。
- “待领取”：条件已经满足，但积分尚未发放，需要点击领取。
- “已完成”：当天或一次性任务已经领取，不能重复领取。

当前任务规则：

- 每日登录：每天 1 分，登录后可领取。
- 记录健康数据：每天 2 分，新增体征后可领取。
- AI 健康咨询：每天 2 分，使用健康助手后可领取。
- 完善健康档案：一次性 20 分，完善基本健康信息后可领取。
- 每日签到：单独使用页面顶部签到按钮，不在普通任务卡中领取。

完成任务后如果仍显示“去完成”，可返回积分商城或刷新页面。任务积分不会仅因进入页面自动到账，看到“待领取”后还要点击。

## 15. 积分商城与兑换记录

### 15.1 浏览商品

路径：顶部“积分中心” → “积分商城”。

商品卡显示图片、名称、类别、说明、所需积分和库存。可以使用类别标签筛选。积分不足或库存为 0 时，兑换按钮不可用。

### 15.2 兑换商品

点击“立即兑换”，在确认框核对商品和所需积分后点击“确认兑换”。

当前普通用户界面一次默认兑换 1 件。兑换成功后会立即扣除积分和商品库存，并生成“待发货”兑换记录。请勿在处理中重复点击。

当前兑换界面没有收货地址填写或地址管理步骤。如果兑换的是需要寄送的实物，应根据平台运营方后续安排补充收货信息；应用助手不能代收地址，也不应要求用户在聊天中发送完整住址。

### 15.3 查看积分明细

路径：顶部“积分中心” → “积分明细” → “积分明细”标签。

明细显示时间、类型或描述、积分增加或扣减以及余额。正数表示获得，负数表示消费或调整。

### 15.4 查看兑换记录

路径：顶部“积分中心” → “积分明细” → “兑换记录”。

兑换状态包括“待发货、已发货、已完成、已取消”。状态由平台工作人员更新，普通用户不能自行修改。当前普通用户页面没有取消兑换按钮；如需取消，应联系平台工作人员。

## 16. 账户管理

### 16.1 账户资料与健康档案的区别

“账户管理”保存头像、用户名、昵称、手机号、性别和生日；“健康档案 → 基本信息”保存身高、体重、血型、病史和紧急联系人。修改一个页面不会自动补齐另一个页面的字段。

### 16.2 修改个人资料

路径：顶部“账户管理” → “个人资料”，或点击右上角头像 → “账户管理”。

用户名只读，不能在此修改。可修改昵称、手机号、性别和生日，点击“保存资料”后生效。

### 16.3 更换头像

在“个人资料”中点击“更换头像”，选择 jpg、jpeg、png、gif 或 webp 图片，文件不超过 2MB。上传成功后还要点击“保存资料”，头像才正式写入账户资料。

如果只上传但没有点击“保存资料”，刷新或离开页面后头像可能不会生效。

### 16.4 修改密码

路径：“账户管理” → “账户安全”。输入原密码和 6 至 32 位新密码，点击“修改密码”。成功后会退出当前登录，需要使用新密码重新登录。

不知道原密码时，不要在“修改密码”中反复尝试；有绑定手机号的用户应在 Web 登录页使用“忘记密码”。

### 16.5 注销账号

路径：“账户管理” → “账户安全” → “注销账号”。

注销会使账号无法继续登录。点击后必须在警告框中再次确认。注销不是普通的退出登录；只想暂时离开时应选择“退出登录”。

执行前应确认是否还需要保留或查询健康档案、咨询记录和积分信息。应用助手不能替用户注销，也不能恢复已注销账号。

## 17. 数据归属、隐私与安全提示

普通用户在健康基本信息、体征、就诊记录、健康报告、健康时间轴、预警、积分明细和医生咨询页面看到的是本人数据。健康时间轴只是把本人已有数据汇总展示，不会把其他用户的数据混入时间轴。不要在公共电脑上保持登录，使用完毕应退出。

健康助手的“结合我的健康档案”开关由用户主动控制。医生咨询附件、咨询报告和用药建议可能含敏感健康资料，查看和发送前应确认当前账号、医生和文件是否正确。

不要向应用使用助手或健康助手发送登录密码、短信验证码、银行卡密码。应用助手不会以“帮您操作”为由索取这些信息。

删除体征、就诊记录或健康报告后，普通用户页面没有恢复入口；删除前应仔细确认。

## 18. 常见问题与标准回答

### 18.1 我找不到某个按钮怎么办？

先查看页面顶部菜单。“健康档案”和“积分中心”需要将鼠标移上去展开子菜单；账户相关功能也可从右上角头像进入。普通用户不会显示管理员的“系统管理”菜单。

### 18.2 昵称和用户名有什么区别？

用户名用于账密登录，注册后普通用户页面不能修改；昵称用于页面展示，可在账户管理中修改。登录失败时要确认输入的是用户名而不是昵称。

### 18.3 为什么收不到验证码？

先确认手机号是 11 位大陆手机号，等待 60 秒，不要频繁重发。验证码 5 分钟有效，同手机号每天有发送次数限制。如果页面提示发送成功但手机始终没有短信，可能是当前部署未接入真实短信服务，应联系平台工作人员。

### 18.4 为什么体征显示异常？

系统按固定阈值自动判断。先核对类型、数值、第二值、单位和测量时间是否填对；填错可点击“编辑”修改。阈值提示不等于诊断，健康问题请使用健康助手或咨询医生。

### 18.5 为什么体征趋势图没有曲线？

同一种指标至少需要 2 条记录。进入“健康档案 → 体征数据”，并确认趋势标签选择了已有记录的指标。

### 18.6 为什么不能生成分析报告？

所选的近 7 天、近 30 天或近 90 天周期内必须至少有 1 条包含有效数值的体征记录。先进入“健康档案 → 体征数据”新增记录，或在健康报告页面选择覆盖已有记录的更长周期。短时间重复提交时还可能提示报告正在生成，请等待当前请求完成。

### 18.7 为什么保存了档案或记录了体征却没有加积分？

这些任务需要手动领取。进入“积分中心 → 积分商城”，找到状态为“待领取”的任务并点击。

### 18.8 为什么签到按钮不见了？

签到按钮在“积分中心 → 积分商城”顶部。当天已签到时按钮会显示“今日已签到”且不可重复点击。

### 18.9 为什么兑换按钮是灰色？

常见原因是积分不足或库存为 0。查看商品所需积分、当前可用积分和库存。兑换成功后积分会立即扣除。

### 18.10 怎样取消兑换？

普通用户页面当前没有取消按钮。请联系平台工作人员；应用助手不能替用户取消。兑换状态可在“积分明细 → 兑换记录”查看。

### 18.11 医生咨询为什么不能再发送？

会话状态如果显示“已结束”，双方都不能继续发送，但仍可查看历史消息。需要新的咨询时返回医生专家库重新选择医生，并以页面实际提示为准。

### 18.12 附件上传失败怎么办？

医生咨询附件只支持图片、PDF 和 Word，单个不超过 10MB。头像只支持 jpg、jpeg、png、gif、webp，且不超过 2MB。检查格式、大小和网络后重试。

### 18.13 为什么保存体征后没有出现预警？

系统只会为达到固定异常阈值且属于支持类型的体征自动创建或更新预警。先核对体征类型、数值、单位和测量时间；正常记录不会创建预警。若是规则升级前的旧记录，可进入“健康预警”点击“重新检测历史体征”查看预览并按需同步。

### 18.14 为什么同类异常只显示一条预警？

同一指标已有未解决预警时，系统会更新原预警的内容、级别、最近触发时间和“本轮已更新次数”，不会每次都新建一条。完全相同的分析结果还会被去重。原预警解决或忽略后，后续新的异常可以创建新一轮预警。

### 18.15 应用助手为什么没有记住刚才的聊天？

Web 顶部“AI 助手”浮窗关闭后会清空当前可见消息，界面没有可恢复的应用助手历史列表。系统启用长期记忆时，可能提取用户明确提供的稳定偏好用于后续回答，但这不等于保存和恢复完整聊天记录；应用助手也不会读取标记为健康私密的长期记忆。重要操作步骤仍应自行记录。

### 18.16 健康助手和应用助手该用哪一个？

平台操作问题用顶部“AI 助手”；症状、疾病、用药和健康风险问题用“健康助手”。医生实时沟通使用“医生咨询”。

### 18.17 删除后能恢复吗？

普通用户页面没有回收站。体征、就诊记录和健康报告删除后不能自行恢复，删除前应确认。

### 18.18 退出登录和注销账号有什么区别？

退出登录只结束本次登录，下次还能登录；注销账号会使账号无法登录。只想暂时离开时不要点“注销账号”。

### 18.19 为什么报告显示“数据不足”？

有体征记录就可以生成报告，但趋势判断至少需要 3 条记录并覆盖 3 个不同测量日。记录不足时，系统仍会展示已有数值和摘要，但综合关注等级可能显示“数据不足”，趋势图或趋势结论也可能不出现。

### 18.20 AI 解读不可用会影响健康报告吗？

不会影响固定规则计算出的统计、异常判定和关注等级。开启 AI 解读但服务不可用时，系统会自动使用规则模板，并在详情中显示生成方式。AI 只整理文案，不改变医学规则结果。

### 18.21 健康时间轴为什么缺少某条记录？

健康时间轴从体征、预警、报告、就诊、医生咨询和用药建议等多个页面汇总最近数据。先到对应来源页面确认记录是否存在；已结束咨询的用药建议只会在医生已经发送建议时显示。某个接口暂时失败或记录超出时间轴最近展示范围时，也可能暂时不出现。

### 18.22 医生咨询结束后为什么还看不到报告、评价或用药建议？

结束会话后需要等待医生提交咨询报告；报告完成前页面显示“等待医生报告”，评价入口也不会开放。用药建议由医生按本次咨询需要单独发送，不是每次咨询都会有。报告完成后仍未刷新时，可重新进入该会话查看。

## 19. 推荐的新用户使用顺序

首次使用建议：

1. 登录后进入“账户管理”，确认昵称、手机号、性别和生日。
2. 进入“健康档案 → 基本信息”，填写身高、体重、血型、病史和紧急联系人。
3. 进入“体征数据”，新增一条真实测量记录并核对单位。
4. 进入“积分中心 → 积分商城”，完成签到并领取已完成任务的积分。
5. 根据需要使用“健康助手”、浏览医生专家或发起医生咨询。
6. 保存体征后留意自动健康预警；如出现预警，从健康助手或医生咨询开始处理，完成后再标记解决。
7. 积累一段时间体征后查看趋势，按需要生成近 7 天、30 天或 90 天健康报告。
8. 进入“健康时间轴”统一查看体征、预警、报告、就诊、咨询和用药建议的处理进度。
